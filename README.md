# Servicios Especiales Pintado Londoño

Landing page corporativa construida con React, Vite y Tailwind CSS para presentar los servicios de transporte especial de pasajeros en Colombia y captar solicitudes de cotización.

## Requisitos

- Node.js 18 o superior
- npm 9 o superior

## Instalar dependencias

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abre en el navegador la URL local que mostrará Vite, normalmente:

```bash
http://localhost:5173
```

## Generar versión de producción

```bash
npm run build
```

## Previsualizar la build

```bash
npm run preview
```

## Publicar en Vercel

1. Sube este proyecto a un repositorio en GitHub, GitLab o Bitbucket.
2. Entra a [Vercel](https://vercel.com/).
3. Crea un nuevo proyecto e importa el repositorio.
4. Vercel detectará automáticamente que es un proyecto Vite.
5. Usa la configuración por defecto:
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. Publica el proyecto.

## Personalizaciones sugeridas

- Reemplazar el logo textual por el logo oficial en formato SVG o PNG optimizado.
- Conectar el formulario a un CRM, correo transaccional o backend si se requiere almacenamiento real de solicitudes.
- Añadir testimonios, cobertura por ciudades o flota si luego desean ampliar la página.
